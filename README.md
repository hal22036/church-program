# Discovery Park Ward Church Program Website

This project is a simple website that displays a digital church program including announcements, calendar events, and program information.

## Features
- Displays church program pages as images
- Responsive layout using HTML and CSS
- Hosted using GitHub Pages

## Technologies Used
- HTML
- CSS
- GitHub Pages
